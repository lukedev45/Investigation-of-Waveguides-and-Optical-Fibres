# Investigation of Waveguides and Optical Fibres

This repository contains the full documentation and report for an undergraduate research project exploring the theory and numerical analysis of optical waveguides, with a focus on optical fibres.

Optical fibres guide light via total internal reflection and support discrete propagation modes. Understanding these modes and how they depend on refractive index profiles is fundamental to optical communication systems and photonics research.

## üß† Key Concepts Covered

The project engages with several foundational topics in waveguide optics and physical modelling:

- **Wave propagation in guided structures:** Light confined in a high-index core surrounded by lower-index cladding.
- **Eigenvalue formulation:** Converting boundary-value problems into discrete mode solutions.
- **Mode analysis:** Identifying how core/cladding profiles influence supported modes.
- **Numerical computation:** Implementing algorithms to approximate eigenmodes relevant for optical fibre design.
